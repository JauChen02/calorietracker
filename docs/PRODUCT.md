# Product — CalorieLog

## What it is

A fast, non-judgmental calorie and macro tracker. The primary promise: log a familiar meal in under 10 seconds.

## Tone

Factual, calm, accessible, not punitive. Copy never references weight loss, guilt, or "cheating." Numbers are informational, never evaluative.

## Launch Scope

The following features are in scope for v1:

| Feature | Notes |
|---|---|
| Manual logging | Log food by name with calories and macros |
| Daily targets | Set personal calorie and macro targets |
| Daily history | View logs day by day |
| Recents | Quick-add recently logged foods |
| Custom foods | Create and save foods with full nutrition data |
| Food search | Search a nutrition database by name |
| Barcode scan | Scan packaged foods with the camera |
| Saved meals | Save and re-log common multi-item meals |
| Offline logging | Log without an internet connection |
| Gentle insights | Simple, non-judgmental weekly summaries |

## Explicit Exclusions

The following are out of scope for v1 and should not be added without a product decision:

- Social feed or sharing
- Workout tracking or exercise logging
- AI photo calorie estimation
- Meal plans or meal planning
- Subscription or paywall
- Medical advice or clinical recommendations
- Apple Health integration
- Android Health Connect integration
